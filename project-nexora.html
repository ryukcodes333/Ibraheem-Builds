Nexora Telegram Bot

A Konosuba-themed Telegram bot with 12 command categories — economy, games, moderation, and more — built with Telegraf and a MongoDB backend.

Node.js
Telegraf
MongoDB

How I Made It

Telegraf made the command routing clean. Each of the 12 categories is its own module that registers its commands on a shared bot instance at startup. The hardest category to build was moderation — Telegram's permission model is group-specific and has a lot of inconsistencies depending on whether you're in a supergroup, regular group, or channel.

javascript 
// Module pattern — each category registers its own commands
const { Telegraf } = require('telegraf');
const bot = new Telegraf(process.env.BOT_TOKEN);

// Load all command categories
require('./commands/economy')(bot);
require('./commands/games')(bot);
require('./commands/moderation')(bot);
// ... 9 more

bot.launch();

The daily reward command was the first thing I built. It was a good test case for the cooldown pattern I'd need everywhere else — check last action timestamp in MongoDB, compute the difference, either pay out or return a "come back in Xh" message.

javascript 
bot.command('daily', async (ctx) => {
  const userId  = ctx.from.id;
  const user    = await User.findOne({ userId });
  const now     = Date.now();
  const cd      = 24 * 60 * 60 * 1000;

  if (user?.lastDaily && now - user.lastDaily < cd) {
    const hrs = Math.ceil((cd - (now - user.lastDaily)) / 3600000);
    return ctx.reply(`⏳ Come back in ${hrs}h, weakling.`);
  }

  const reward = Math.floor(Math.random() * 500) + 100;
  await User.updateOne({ userId }, {
    $inc: { coins: reward }, $set: { lastDaily: now }
  }, { upsert: true });

  ctx.reply(`✨ Aqua blesses you with *${reward} coins*!`,
    { parse_mode: 'Markdown' });
});

The Konosuba theming was actually the most fun part. Every response message references characters and lore from the anime. I wrote a helper that wraps each category's responses with a random in-character line depending on the command type — Aqua for blessings, Kazuma for sarcastic comments, Megumin for anything explosive.
