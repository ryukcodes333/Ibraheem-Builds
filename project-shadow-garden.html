Shadow Garden Bot

A full-featured WhatsApp bot with an in-chat economy, Pokémon catching and battling, and RPG mechanics — all running over Baileys with MongoDB persistence.

Node.js
Baileys
MongoDB

How I Made It

Started with Baileys — a Node.js library that reverse-engineers the WhatsApp Web API. The hardest part wasn't the bot logic, it was keeping the connection alive. WhatsApp aggressively disconnects sessions, so I had to build a reconnect handler that re-authenticates from saved credentials without prompting a new QR scan every time.

javascript 
// Auto-reconnect on disconnect
client.on('connection.update', async ({ connection, lastDisconnect }) => {
  if (connection === 'close') {
    const code = lastDisconnect?.error?.output?.statusCode;
    const shouldReconnect = code !== DisconnectReason.loggedOut;
    if (shouldReconnect) startBot(); // recursive restart
  }
});

// Save credentials on update
client.on('creds.update', saveCreds);

The economy system stores every user's balance and inventory in MongoDB. Each command handler reads and writes atomically to prevent race conditions when two users trigger commands at the same time.

javascript 
async function handleBalance(sock, msg) {
  const jid  = msg.key.remoteJid;
  const user = await User.findOneAndUpdate(
    { jid },
    { $setOnInsert: { balance: 100, inventory: [] } },
    { upsert: true, new: true }
  );
  await sock.messages.send(jid, {
    text: `💰 Balance: *$${user.balance}*`
  });
}

The Pokémon system assigns each catch a random species from a local JSON dataset. Battles run a turn-based loop comparing attack and defense values with a small RNG factor so no fight is fully predictable. The hardest edge case was handling users who rage-quit mid-battle — the system needs to clean up the battle state from the database or it blocks both users forever.

javascript 
// Random Pokémon encounter
function spawnPokemon() {
  const pool = pokemonData.filter(p => p.rarity === 'common');
  const poke = pool[Math.floor(Math.random() * pool.length)];
  return {
    ...poke,
    hp: poke.baseHp + Math.floor(Math.random() * 20),
    caught: false
  };
}
